# FasterHQ Homebrew Tap

Install the **`faster`** CLI — the chat-first terminal client for a Faster box:

```
brew tap fasterhq/tap
brew install faster
```

<sub>(one-line equivalent: `brew install fasterhq/tap/faster`)</sub>

Then:

```
faster login --org <id>
faster chat
```

Signed **Developer ID** universal macOS binaries (arm64 + x86_64) are attached to this repo's
releases; the source lives in a separate private repo. Upgrade with `brew upgrade faster`.
